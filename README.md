# ICSI-project

---

**How to run it:**

1. Create a new Java class named exactly `RockPaperScissors`
2. Paste this code in, replacing everything in the file
3. Run it — the console will prompt you to type your choice
4. Type `rock`, `paper`, or `scissors` and hit Enter

-----

**What it does:**
- You type your pick → computer randomly picks one of the three → program compares them and prints who won → program keeps score → first to win 8 round (best of 15) wins the game
- Input is case-insensitive (`Rock` and `rock` both work)
- If you type something invalid, it tells you and the round restarts
- Ties do not count as a round — if there is a tie the round will restart
